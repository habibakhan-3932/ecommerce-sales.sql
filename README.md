# E-Commerce Sales Database (MySQL Project)
This project is a beginner-friendly MySQL database system for an E-Commerce platform.
It includes a complete schema, CSV datasets, and a list of SQL questions that can be performed on the data.

The project helps you understand how an online store manages customers, products, orders, and payments.
 
 ->2.Datasets Used
These CSV files were imported into MySQL:

customers.csv

products.csv

orders.csv

order_items.csv

payments.csv

Each dataset contains realistic e-commerce data such as customer details, product information, order history, and payments.

3. Questions Performed in This Project
Below are the SQL questions included in the project (queries not shown):

Display all customers.

List all products with their category and price.

Show all orders placed in 2024.

Find total number of orders.

Show products with stock less than 50.

Find the highest-selling category.

Find the total amount spent by each customer.

List orders along with customer names.

Show top 5 most expensive products.

Calculate total revenue from all orders.

Find number of orders for each payment method.

Show products that were never ordered.

List customers who placed more than 3 orders.

Find the most popular product (highest quantity sold).

Display all orders along with total items in each order.

These questions help you practice:

JOINs

GROUP BY

Aggregation functions

Filtering & sorting

Basic analysis
